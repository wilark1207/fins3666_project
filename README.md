# Project Environment Setup

  Anaconda is a Python distribution that makes it easy to manage environments and dependencies in one place.

### 1.Install Anaconda (or Miniconda)
1. Go to [https://www.anaconda.com/download](https://www.anaconda.com/download) and install Anaconda for your system.  
   (Alternatively, use [Miniconda](https://docs.conda.io/en/latest/miniconda.html) for a lighter version.)
2. Open **Anaconda Prompt** (Windows) or **Terminal** (Mac/Linux).

### 2. Navigate to the project folder

cd to whatever path

### 3. Create environment

conda env create -f environment.yml

### 4. Activate environment

conda activate fins3666
